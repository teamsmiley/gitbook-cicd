## cors 설정

bucket &gt;&gt; permission &gt;&gt; cors 설정

```javascript
[
  {
    AllowedHeaders: ['Authorization'],
    AllowedMethods: ['GET'],
    AllowedOrigins: ['https://www.example.com', 'https://staging.example.com'],
    ExposeHeaders: [],
    MaxAgeSeconds: 3000,
  },
];
```
